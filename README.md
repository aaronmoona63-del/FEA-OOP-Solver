# FEA-OOP-Solver
# 三维高性能面向对象有限元求解器 (OOP-FEM Solver)

本项目是一个基于现代 Fortran 语言开发的底层三维实体有限元计算引擎，旨在为大规模结构力学分析提供高效、稳定且可扩展的数值求解方案。有别于传统的面向过程开发模式，本项目深度应用了面向对象编程（OOP）思想，通过构建高度抽象的矩阵基类与求解器多态接口，实现了有限元底层数据结构与物理计算逻辑的解耦，大幅提升了代码的二次开发能力与长期维护性。

在稀疏矩阵管理方面，本引擎原生开发了完备的存储生态。系统支持 COO、CSR 以及 Skyline（变带宽）等多种稀疏矩阵格式，并内置了极低开销的跨格式内存转换算法。这一设计不仅满足了总刚度矩阵在装配阶段对随机写入的高要求，同时也为后续方程求解阶段的高速连续内存访问提供了底层保障。

求解核心模块采用了灵活的“双引擎”架构以应对不同规模的计算需求。对于极高精度的需求，系统通过专用接口深度集成了 Intel MKL PARDISO 商业级直接求解器，充分利用多核并行优势进行超大规模矩阵的精确分解；而在面临受限内存环境或超大自由度挑战时，系统则可无缝切换至自研的预处理共轭梯度法（PCG）迭代求解器。该迭代求解器针对三维计算特点进行了深度优化，有效规避了内存溢出（OOM）问题。

本项目的有限元装配层实现了标准的三维八节点六面体等参单元（Hex8），集成了高斯数值积分、刚度矩阵组装以及多种边界条件的精确施加（如 Dirichlet 约束的置零置一法与乘大数法）。在严苛的基准压测中，本引擎成功完成了 13.5 万单元（约 43.5 万自由度）的带孔平板拉伸与悬臂梁复杂应力分析。通过自建的 VTK 结果导出模块，其全场位移数据在 ParaView 中的三维可视化分布与 Abaqus 商业软件的计算结果高度吻合，充分验证了求解算法的准确性与鲁棒性。

## 项目结构说明

整个代码库被合理划分为核心源码、测试验证与辅助工具三个维度。`src/` 目录下包含了三大核心库：`MatrixLayer` 负责所有数据结构的管理与转化，`SolverLayer` 集中实现了直接法与迭代法求解算法的调度，`assemble_boundary_layer` 统筹了单元形函数计算、全局刚度矩阵拼接与边界载荷处理。`tests/` 目录中涵盖了从底层单元测试到物理力学验证（如纯弯曲、偏心拉伸）的数十个算例，所有算例均由 CMake 脚本实现自动化构建与回归测试。此外，`scripts/` 目录提供了系列 Python 自动化工具，涵盖了从十万级参数化网格生成到后处理数据曲线拟合的完整工作流。

## 编译与运行指南

本项目采用 CMake 作为全局构建系统，依赖 Intel Fortran 编译器与 Intel MKL 核心数学库。编译时，请在项目根目录新建构建文件夹并执行 CMake 配置，推荐使用并行编译指令（如 `make -j4`）以加速构建过程。编译完成后，可直接在构建目录内执行相应的力学基准测试模块（如 `./tests/test_benchmark_iter`）以复现大规模分析结果。
